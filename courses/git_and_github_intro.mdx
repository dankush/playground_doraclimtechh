## מה תקבלו בסוף הקורס?

- **הבנה מעשית של Git** – עבודה עם מערכת ניהול גרסאות לניהול קוד
- **שימוש ב-GitHub לפרויקטים שיתופיים** – העלאת קוד, ניהול משימות וביקורת קוד
- **יכולות שיתוף פעולה** – עבודה בצוותים עם Pull Requests ו-Code Reviews
- **פרויקט מסכם** – יצירת פרויקט אקלים פתוח ב-GitHub

---

## משך הקורס

- **10 מפגשים** (שעה וחצי כל מפגש)
- **סה"כ 15 שעות**
- **כולל תרגולים מעשיים ופרויקט מסכם**

---

## למי זה מתאים?

- לכל מי שמעוניין ללמוד **Git ו-GitHub מאפס** ולשפר את שיטות העבודה שלו
- מפתחים, סטודנטים ואנשי סביבה שרוצים **לנהל פרויקטים בצורה מקצועית**
- כל מי שמעוניין **להצטרף לקהילות קוד פתוח** ולעבוד על פרויקטים שיתופיים

---

## סילבוס מפורט

| מפגשים | נושא | תיאור |
|---------|------|--------|
| **1** | מבוא ל-Git ולניהול גרסאות | למה צריך מערכת ניהול גרסאות? מושגי יסוד: Commit, Repository, Branch, Merge |
| **2** | התקנה ושימוש בסיסי | התקנת Git, יצירת Repository מקומי, ביצוע Commit-ים והיסטוריית גרסאות |
| **3** | עבודה עם GitHub | יצירת חשבון, יצירת Repository ב-GitHub, העלאת קבצים ראשונה |
| **4** | עבודה עם Branches | יצירת ענפים, מעבר בין Branch-ים, מיזוג שינויים (Merge) |
| **5** | שיתוף פעולה על פרויקטים | עבודה עם Pull Requests, Code Review ושילוב שינויים |
| **6** | פתרון קונפליקטים | הבנת Merge Conflicts, שימוש בכלי פתרון קונפליקטים |
| **7** | עבודה עם Git Hooks | אוטומציה באמצעות Git Hooks, יצירת כללים בפרויקטים שיתופיים |
| **8** | GitHub Actions ואוטומציה | יצירת תהליכי CI/CD בסיסיים עם GitHub Actions לפרויקטים אקלים |
| **9** | ניהול פרויקטים סביבתיים | יצירת פרויקט אקלים עם נתונים אמיתיים, עבודה עם Open Data |
| **10** | פרויקט מסכם | עבודה מעשית, יצירת Repository שיתופי לפרויקט סביבתי, הצגת תוצרים |

---

## פרויקט מסכם – **"מעקב אחרי פליטות פחמן"**

- עבודה על פרויקט אמיתי של **מעקב אחר פליטות פחמן לפי אזורים גיאוגרפיים**
- שימוש ב-Git לניהול קוד ושיתוף נתונים
- שילוב מידע ממקורות פתוחים, ניתוח נתונים והצגת גרפים
- הצגת הפרויקט ב-GitHub ושיתוף הקהילה

---

## למה כדאי להירשם?

- **קורס פרקטי וממוקד** – לימוד Git ו-GitHub בשיטה מעשית
- **עבודה על פרויקט אמיתי** – שילוב עולם הפיתוח עם אתגרי הסביבה
- **מיומנויות נדרשות לתעשייה** – כלים שכל מתכנת ופעיל סביבתי צריכים
- **ליווי מקצועי** – תמיכה לאורך כל הקורס, כולל קהילה ושיתוף פעולה
