## סילבוס מפורט

### מודול 1: יסודות תכנות – מבוא לעולם Python ויישומים סביבתיים (7 שיעורים)

1. **שיעור 1: היכרות עם המיזם והסביבה הפיתוחית**
   - הצגת חזון "דור אקלים טק" ושילוב טכנולוגיה באתגרים סביבתיים.
   - התקנת <span dir="ltr">Python</span> והיכרות עם <span dir="ltr">Google Colab</span>, <span dir="ltr">Jupyter Notebook</span> ו־<span dir="ltr">NotebookLM</span>.
   - שימוש ב־<span dir="ltr">Discord</span> ו־<span dir="ltr">GitHub</span> לשיתוף ידע וקוד.
   - **תרגיל**: כתיבת תוכנית "<span dir="ltr">Hello, World!</span>" והצטרפות לשרת <span dir="ltr">Discord</span>.

2. **שיעור 2: משתנים, סוגי נתונים וקלט/פלט – דוגמה אקלימית**
   - משתנים בסיסיים (<span dir="ltr">int</span>, <span dir="ltr">float</span>, <span dir="ltr">str</span>, <span dir="ltr">bool</span>) ופונקציות <span dir="ltr">input()</span>/<span dir="ltr">print()</span>.
   - יישום בדוגמאות סביבתיות (מדידת טמפרטורה, לחות).
   - **תרגיל**: סקריפט המדמה קלט נתוני סביבה ומציג התראה אם הערך חריג.

3. **שיעור 3: משפטי תנאי ולולאות – שליטה בזרימת הקוד**
   - מבני בקרה <span dir="ltr">if</span>/<span dir="ltr">else</span>, לולאות <span dir="ltr">for</span>/<span dir="ltr">while</span>.
   - דוגמאות מעשיות מעולם הסביבה (אחוז זיהום, טמפרטורות שבועיות).
   - **תרגיל**: כתיבת קוד הבודק ערכים ומציג התראה במקרה של חריגה.

4. **שיעור 4: עבודה עם קבצים (CSV, JSON)**
   - קריאה וכתיבה של קבצים בפייתון.
   - טעינת נתוני אנרגיה מתחדשת וניתוחם.
   - **תרגיל**: יצירת דוח מסכם מצריכת חשמל עולמית.

5. **שיעור 5: עבודה עם API ושליפת נתונים חיצוניים**
   - קריאה ל־API לקבלת נתוני איכות אוויר בזמן אמת.
   - המרת הנתונים למבנה מתאים וניתוחם.
   - **תרגיל**: כתיבת סקריפט שמפיק דוח איכות אוויר.

6. **שיעור 6: פונקציות ומודולריות – כתיבת קוד נקי ואפקטיבי**
   - הגדרת פונקציות, פרמטרים והחזרת ערכים.
   - תרגול על נתוני סביבה (חישוב ממוצעים, זיהוי חריגות).
   - **תרגיל**: עבודה בקבוצות ליצירת סקריפט מלא המשקף את כל הנושאים עד כה.

7. **שיעור 7: תרגיל סיכום מודול 1**
   - פרויקט משולב ב־<span dir="ltr">Colab</span>: משתנים, תנאים, לולאות, קבצים, API ופונקציות.
   - **תרגיל**: עבודה קבוצתית – יצירת סקריפט מסכם עם עיבוד נתונים ותיעוד קוד.

### מודול 2: ניהול פרויקטים וכלים נוספים (4 שיעורים)

1. **שיעור 1: יצירת WBS – חלוקת פרויקט ניתוח פליטות גזי חממה**
   - הכרת מושג ה־WBS ופירוק פרויקט למרכיביו.
   - שימוש בכלים כמו <span dir="ltr">ClickUp</span>, <span dir="ltr">Trello</span>, <span dir="ltr">Monday</span>.
   - **תרגיל**: עבודה בקבוצות ליצירת WBS לפרויקט לדוגמה.

2. **שיעור 2: עבודה עם פלטפורמות ענן**
   - סקירה על אחסון ועיבוד נתונים בענן (<span dir="ltr">AWS</span>, <span dir="ltr">Google Cloud</span>).
   - ביצוע שאילתות בסיסיות לניתוח נתונים.
   - **תרגיל**: העלאת נתוני אקלים לשרת ענן וביצוע ניתוחים ראשוניים.

3. **שיעור 3: יסודות ניהול פרויקטים – Agile, Scrum ושימוש ב־ClickUp**
   - הכרת פילוסופיית <span dir="ltr">Agile</span> ושיטות <span dir="ltr">Scrum</span>.
   - יצירת לוחות, רשימות וכרטיסים (<span dir="ltr">Kanban</span>) ב־<span dir="ltr">ClickUp</span>.
   - **תרגיל**: הקמת פרויקט ב־<span dir="ltr">ClickUp</span> עם חלוקת משימות ותיעוד.

4. **שיעור 4: תכנון וביצוע פרויקט**
   - הגדרת מטרות <span dir="ltr">SMART</span>, חלוקת תפקידים ומשימות.
   - ניהול ספרינטים ועדכון סטטוסים.
   - **תרגיל**: עבודה קבוצתית – מיפוי אתגר סביבתי והטמעת מודל ML בפרויקט.

### מודול 3: AI ניתוח נתונים מתקדם וכלים נוספים (4 שיעורים)

1. **שיעור 1: מאחורי הקלעים של AI ו־LLM – הבנה ושימוש נכון**
   - איך מודלים כמו <span dir="ltr">transformers</span> פועלים (<span dir="ltr">tokenization</span>, <span dir="ltr">attention</span>).
   - תהליך אימון מודלים (<span dir="ltr">pre-training</span>, <span dir="ltr">fine-tuning</span>), אתגרים (<span dir="ltr">bias</span>, משאבי <span dir="ltr">GPU</span>).
   - שימושי LLM מול מודלים אחרים (לזיהוי תמונה, ניתוח נתונים וכו').
   - **תרגיל**: "<span dir="ltr">Deep Research</span>" קצר – חיפוש מידע סביבתי והצגת תובנה חדשה.

2. **שיעור 2: היכרות ושימוש ב־NotebookLM ככלי למידה מתקדם**
   - מהו <span dir="ltr">NotebookLM</span> – עיבוד מסמכים, סיכומים ושאלות על בסיס התוכן שהועלה.
   - **תרגיל**: הדגמת שימוש ב־<span dir="ltr">ChatGPT</span> וב־<span dir="ltr">NotebookLM</span> לזיהוי מגמות בנתונים אקלימיים.

3. **שיעור 3: מבוא לכלי Groq, Gimini ו־DeepSeek – חיפוש תובנות וייעול ניתוחים**
   - שימוש בכלים מתקדמים לחיפוש תיעוד, דוגמאות קוד ואופטימיזציה של מודלים.
   - **תרגיל**: שיפור מודלים קיימים בעזרת כלים אלו.

4. **שיעור 4: יצירת מודלים חזותיים עם Google AI Studio ו־Julius**
   - בניית מודלים גרפיים להצגת נתוני אקלים.
   - שילוב <span dir="ltr">Julius</span> ליצירת גרפים.
   - **תרגיל**: בניית גרף אינטראקטיבי המציג מגמות סביבתיות.

### מודול 4: פרויקט גמר – פיתוח, יישום והצגה (3 שיעורים)

1. **שיעור 1: התחלת עבודה על פרויקט הגמר המשולב**
   - הגדרת נושא פרויקט בקבוצות (3 משתתפים).
   - כתיבת תוכנית עבודה ב־<span dir="ltr">ClickUp</span> והתחלת כתיבת קוד וניתוח נתונים.

2. **שיעור 2: עבודה על הפרויקט – סיום, תיקונים והכנות להצגה**
   - ליטוש קוד, בדיקות ותיקון באגים.
   - הכנת מצגת מסכמת ותיעוד ב־<span dir="ltr">ClickUp</span>.

3. **שיעור 3: הצגת פרויקט הגמר וסיכום המיזם**
   - הצגת הפרויקטים והפקת לקחים.
   - סיכום כולל וחלוקת תעודות סיום.
